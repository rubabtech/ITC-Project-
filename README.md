# University Parking Management System

A University Parking Management System built in C++ for ITC (Introduction to Computing) course.

## Features
- Park, remove, and search vehicles (Bike, Car, Van, Scooty)
- 10 parking slots with real-time status
- Automatic fee calculation per vehicle type
- Uses only `#include<iostream>` — no string library

## Concepts Used
- char arrays (no string library)
- for loop, while loop, do-while loop
- cin / cout
- Arrays
- Functions & switch-case
